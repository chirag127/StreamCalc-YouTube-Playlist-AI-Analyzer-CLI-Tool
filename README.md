# StreamCalc: YouTube Playlist AI Analyzer

[![CI](https://github.com/jules-agent/StreamCalc-YouTube-Playlist-AI-Analyzer-CLI-Tool/actions/workflows/ci.yml/badge.svg)](https://github.com/jules-agent/StreamCalc-YouTube-Playlist-AI-Analyzer-CLI-Tool/actions/workflows/ci.yml)
[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

An AI-powered CLI tool to analyze and calculate the total length of YouTube playlists, now reborn as a sleek, frontend-only web application.

## ‚ú® Features

- **Instant Playlist Analysis:** Paste a YouTube playlist URL and get the total duration in seconds.
- **Frontend-Only:** No backend, no servers, no hassle. Runs entirely in your browser.
- **User-Provided API Keys:** Securely use your own API keys for various services (though not required for the core functionality).
- **Modern UI:** Built with Vite, React, TypeScript, and Tailwind CSS for a fast, responsive, and beautiful user experience.
- **Spatial-Adaptive Design:** A modern UI that adapts to your screen size and orientation.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jules-agent/StreamCalc-YouTube-Playlist-AI-Analyzer-CLI-Tool.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Open your browser to `http://localhost:5173`
