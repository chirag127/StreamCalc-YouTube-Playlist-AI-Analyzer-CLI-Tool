Information We Do Not Collect

This application is designed to be fully client-side. No user data — including YouTube playlist URLs, timestamps, or any other input — is sent to any server or stored anywhere beyond your browser.

How the App Works

The application uses the YouTube Data API (via public endpoints) to fetch playlist metadata directly in your browser. All processing — including duration calculation using AI logic inspired by ChatGPT — happens locally.

The logic and interface are open-source and available on GitHub for full transparency.

The app relies on the following third-party services:
